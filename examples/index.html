@pinecall/web — framework-agnostic Web Components (weather demo agent)

① <pinecall-orb>

Voice orb launcher (bottom-right). opens decides what a click does.

② <pinecall-modal>

Glass call modal. visual="wave" uses real audio metrics; keyboard button → live transcript + text-during-call.

③ <pinecall-chat>

Docked chatbox. Text-first; the call button escalates to a WebRTC voice call (conversation continues).
